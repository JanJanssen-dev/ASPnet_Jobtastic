# ASPnet_Jobtastic# Blazor Todo App

Dies ist eine Blazor Todo App, die mit .NET 8 entwickelt wurde. Diese Anwendung ermöglicht es Benutzern, Aufgaben zu erstellen, zu bearbeiten und zu löschen.

## Voraussetzungen

- .NET 8 SDK
- Microsoft SQL Server

## Installation

1. Klone das Repository:

2. Stelle sicher, dass du .NET 8 SDK installiert hast. Du kannst es [hier](https://dotnet.microsoft.com/download/dotnet/8.0) herunterladen.

3. Stelle sicher, dass du einen Microsoft SQL Server installiert und konfiguriert hast.

## Konfiguration

1. Öffne die `appsettings.json`-Datei im Projektverzeichnis.

2. Ändere den Connection String für die Datenbank:
**Hinweis:** Diese Anwendung funktioniert nur mit Microsoft SQL Server. Wenn du eine andere Datenbank verwenden möchtest, musst du alle Models anpassen.

## Ausführen der Anwendung

1. Navigiere zum Projektverzeichnis und führe den folgenden Befehl aus:
2. Öffne deinen Browser und gehe zu `https://localhost:5001`, um die Anwendung zu verwenden.

## Features

- Aufgaben erstellen
- Aufgaben bearbeiten
- Aufgaben löschen
- Aufgaben als erledigt markieren
